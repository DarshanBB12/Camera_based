# Camera Sidebar Project (HTML & CSS)

This project is a Photography Website Sidebar created using HTML and CSS.
It is a front-end layout featuring a dark camera-themed background and a sliding sidebar menu, built to practice layout design, styling, and modern UI development.


# Project Description

The goal of this project is to create a clean and visually appealing photography interface with a full-screen background and a toggle sidebar navigation.
It focuses on building a responsive layout and implementing interactive UI using pure CSS without JavaScript.

# Features

~Full-screen camera background
~Sidebar navigation (Gallery, Shortcuts, Exhibits, Events, Store, Contact, Feedback)
~Toggle sidebar using pure CSS (checkbox method)
~Social media icons section
~Smooth sidebar slide animation
~Clean and modern dark theme

# Technologies Used

~HTML5
~CSS3
~Flexbox & Positioning
~Font Awesome
~Google Fonts
